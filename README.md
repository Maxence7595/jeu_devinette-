# Jeu de Devinette - Nombre Mystere

## Description
Un jeu simple en Python ou le joueur doit deviner un nombre mystere choisi aleatoirement par l'ordinateur.

## Fonctionnalites
- Menu principal avec options
- Choix du niveau de difficulte (facile, moyen, difficile)
- Systeme d'indices (plus grand / plus petit)
- Compteur de tentatives
- Affichage du score final

## Comment lancer le jeu

### Prerequis
- Python 3.x installe sur votre machine

### Execution
Ouvrez un terminal et tapez:

python main.py
